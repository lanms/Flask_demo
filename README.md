   # demo-code

   一个基于python3.6 flask demo

- 安装依赖
- 运行项目
- 运行测试
- 配置开发环境的数据库

   # 安装依赖

   新建一个python3.6的环境，进入环境运行下面命名

   ```
   $ pip install -r requirements.txt
   ```

   # 运行项目

   本地运行， 环境变量请看 settings.py 中 TestConfig

   ```
       $ python develop.py
   ```

   ```
       $ python application.py
   ```

# 运行测试

  在你的环境安装 tox，进入项目跟路径 运行

```
$ tox
```
